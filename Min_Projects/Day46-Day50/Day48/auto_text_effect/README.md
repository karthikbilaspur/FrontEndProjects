# DEMO Website

A responsive website showcasing various features and functionalities, built using HTML, CSS, and JavaScript.

## Features

* Responsive design for mobile, tablet, and desktop devices
* Animated navigation with dropdown menu
* Smooth scrolling to sections
* Auto-text-effect in the hero section
* Read more functionality for descriptions
* Contact form with basic styling
* Social media links in the footer

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

* `index.html`: Main HTML file
* `styles.css`: Custom CSS styles
* `script.js`: JavaScript code
