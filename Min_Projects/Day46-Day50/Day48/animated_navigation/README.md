# Animated Navigation Website

A responsive website with animated navigation, built using HTML, CSS, JavaScript, and Bootstrap.

## Features

* Responsive design for mobile, tablet, and desktop devices
* Animated navigation with dropdown menu
* Smooth scrolling to sections
* Form validation and submission
* Social media links
* Schema markup for organization

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* Font Awesome icons

## Project Structure

* `index.html`: Main HTML file
* `styles.css`: Custom CSS styles
* `script.js`: JavaScript code
* `favicon.ico`: Favicon image
* `image.jpg`, `image-small.jpg`, `image-medium.jpg`: Images used in the website
