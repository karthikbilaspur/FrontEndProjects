# Split Landing Page Project

A simple and responsive split landing page built with HTML, CSS, and JavaScript.

## Overview

This project is a simple and responsive split landing page that showcases a website's services, features, and contact information. The page is designed to be visually appealing and easy to navigate, with a clean and modern design.

## Features

* Responsive design for mobile, tablet, and desktop devices
* Split landing page layout with hero section, services, testimonials, and contact form
* Customizable and easy to update
* Includes Font Awesome icons and AOS (Animate On Scroll) library for animations

## Technologies Used

* HTML5
* CSS3 (with media queries for responsiveness)
* JavaScript (with ES6 syntax)
* Font Awesome icons
* AOS (Animate On Scroll) library

## Setup and Installation

1. Download the project files
2. Open the `index.html` file in a web browser to view the project
3. Edit the HTML, CSS, and JavaScript files to customize the project

## Usage

* Update the images and icons to match your brand
* Add or remove sections as needed
* Customize the styles and layout in the CSS file
* Add custom JavaScript code to enhance the project's functionality

## Project Structure

* `index.html`: The main HTML file for the project
* `style.css`: The CSS file for styling the project
* `script.js`: The JavaScript file for adding interactivity to the project
* `favicon.ico`: The favicon for the project
* `image.jpg`: Sample image used in the project

## Tips and Tricks

* Use a code editor like VS Code to edit the project files
* Use the browser's developer tools to inspect and debug the project
* Test the project on different devices and browsers to ensure responsiveness
