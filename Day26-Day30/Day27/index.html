Anagram Game

Test your vocabulary and word skills with our Anagram Game!

Game Rules:

1. Choose a category and difficulty level.

2. Guess the scrambled word.

3. You have a limited time to guess the word.

4. Use hints to help you guess the word.

5. Score points for each correct guess.

How to Play:

1. Select a Category:

Choose from countries, animals, or food.

2. Select a Difficulty:

Choose from easy, medium, or hard.

3. Start the Game:

Click the start button to begin.

4. Guess the Word:

Type your guess in the input field.

5. Check Your Answer:

Click the check button to see if you're correct.

Hints:

Use the hint button to get a hint about the word.

Leaderboard:

See how you rank against other players.
