Interactive Quiz Game

Quiz Game Overview

The Interactive Quiz Game is a fun and engaging way to test your knowledge on various topics. The game consists of multiple questions, each with a unique category and difficulty level. The questions are designed to challenge your understanding and encourage you to learn more.

Question Format

Each question will be presented in one of the following formats:

  • MCQ (Multiple Choice Question)

    : Choose the correct answer from the given options.

  • Fill-up

    : Fill in the blank with the correct answer.

  • Simple Question

    : Answer the question with a numerical value.

Marking Scheme

Each question carries a certain number of marks, which will be awarded if you answer correctly. The marks are as follows:

  • Easy Questions

    : 1 mark

  • Difficult Questions

    : 2 marks

  • Hard Questions

    : 3 marks

If you use a hint, 1 mark will be deducted from your score.

Rules and Regulations

To play the game, please follow these rules:

  • Time Limit
    : You have 60 seconds to answer each question.
  • Answer Submission
    : Submit your answer using the "Submit" button.
  • Next Question
    : Click "Next" to move to the next question.
  • Skip Question
    : Click "Skip" to skip the current question.
  • Hint
    : Click "Hint" to get a hint for the current question.
