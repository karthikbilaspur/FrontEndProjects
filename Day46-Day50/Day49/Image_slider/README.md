# Image_Slider

This is a simple image gallery application that allows users to browse and download high-quality nature images. The application uses the Unsplash API to fetch images and provides features like search, navigation, download, and sharing.

Features
Browse nature images
Search images by keyword
Download images
Share images on social media
Toggle image info
Fullscreen mode
Auto-change images every 5 seconds

Technologies Used
HTML5
CSS3
JavaScript (ES6+)
Bootstrap 5
Unsplash API
Getting Started

API Key
To use this application, you need to have an Unsplash API key. You can get one for free by creating an account on Unsplash.
