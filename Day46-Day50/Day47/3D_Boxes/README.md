# 3D Flip Box Animation

A simple and responsive 3D flip box animation using HTML, CSS, and JavaScript.


## Features

* 3D flip box animation
* Responsive design
* Customizable flip direction (horizontal, vertical, diagonal)
* Customizable trigger (hover, click)
* Smooth scrolling to anchors
* Animated on load

## Usage

1. Add the `flip-box` class to your HTML element
2. Add the `data-flip-direction` attribute to specify the flip direction (horizontal, vertical, diagonal)
3. Add the `data-flip-trigger` attribute to specify the trigger (hover, click)

Example:
```html
<div class="flip-box" data-flip-direction="horizontal" data-flip-trigger="hover">
    <!-- your content here -->
</div>
Customization

You can customize the styles by editing the style.css file
You can add more flip directions and triggers by editing the script.js file
