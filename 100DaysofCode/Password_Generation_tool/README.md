# Password Generator

A secure and reliable password creation tool that generates strong and unique passwords.

## Features

Simple and Advanced Password Generation**: Generate passwords with customizable length, uppercase, numbers, and special characters.
Password Strength Meter**: Evaluate the strength of generated passwords.
Password History**: Keep track of previously generated passwords.
Dark Mode**: Switch to a dark theme for comfortable use in low-light environments.
Language Support**: Available in multiple languages, including English, Spanish, French, Hindi, Kannada, and Marathi.

## Usage

1.Select a password generator (Simple or Advanced) from the dropdown menu.
2.Customize password settings (length, uppercase, numbers, special characters) as needed.
3.Click the "Generate" button to generate a password.
4.Copy the generated password to your clipboard using the "Copy" button.
5.Evaluate the password strength using the strength meter.
6.View previously generated passwords in the password history section.
