# Kboard Dashboard

A simple and intuitive dashboard for monitoring your app's performance and engagement metrics.

## Features

* Displays key metrics such as users, posts, and engagements
* Visualizes traffic and engagement data with charts
* Supports dark mode for easy viewing
* Responsive design for mobile devices

## Usage

1. Open your web browser and navigate to `http://localhost:3000`
2. View your dashboard metrics and charts
3. Toggle dark mode by clicking the theme toggle button
