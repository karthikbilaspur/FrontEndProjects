# Programming Languages Accordion

A multi-level accordion showcasing programming languages and their main features.

## Table of Contents

* [Features](#features)
* [Usage](#usage)
* [Technologies Used](#technologies-used)

## Features

* Multi-level accordion with smooth toggling and closing of other accordions on the same level
* Responsive design for smaller screens
* Smooth scrolling for anchor links
* Easy to customize and extend

## Usage

* Click on an accordion header to toggle its content
* Click on an anchor link to smoothly scroll to the target element

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
