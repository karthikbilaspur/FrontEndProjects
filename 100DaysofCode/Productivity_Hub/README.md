# My Smart Productivity Hub

## 🚀 Overview

Welcome to My Smart Productivity Hub! This is an allinone web application designed to boost your efficiency and help you manage your daily life with ease. It combines essential productivity tools into a single, intuitive interface, built using modern HTML, CSS, and vanilla JavaScript.

Whether you need to focus with a Pomodoro timer, keep track of your tasks, jot down quick notes, or organize your schedule with a calendar, this hub has you covered.

## ✨ Features

   Pomodoro Timer:
       Customizable Pomodoro, Short Break, and Long Break durations.
       Start, Pause, and Reset functionality.
       Audio notification upon session completion.
       Browser tab title updates to show current time.
       Seamless autoswitching between Pomodoro and break modes.
   ToDo List:
       Add, Mark Complete/Incomplete, and Delete tasks.
       Edit existing tasks.
       Option to set any ToDo item as the current Pomodoro focus.
       All tasks are saved automatically to your browser's local storage.
   Notes App:
       Create, view, and Delete notes.
       Inline editing for note titles.
       Full content editing for notes via dedicated input area.
       All notes are saved automatically to your browser's local storage.
   Dynamic Calendar:
       View monthly calendar with current month/year display.
       Navigate between months.
       Highlighting for the current day.
       Add, View, Edit, and Delete events for any day.
       Event details and management are handled through an interactive modal.
       Events are saved automatically to your browser's local storage.
   UserFriendly Interface:
       Clean, responsive design using HTML, CSS (Flexbox/Grid), and Font Awesome icons.
       Intuitive navigation between different productivity tools.

## 🛠️ Technologies Used

   HTML5: For the core structure of the application.
   CSS3: For styling, layout, and responsiveness.
   JavaScript (Vanilla): For all interactive logic, data management, and dynamic content.
   Local Storage API: For persistent data storage (ToDos, Notes, Calendar Events, Pomodoro Settings).
   Font Awesome: For a wide range of vector icons.

## 🚀 Getting Started

To run this project locally, follow these simple steps:

1.Clone the repository (or download the files):

    Alternatively, simply copy the `index.html`, `style.css`, and `script.js` files into a new folder on your computer.

2.Ensure all files are in the same directory:
       `index.html`
       `style.css`
       `script.js`

3.Optional: Add a Custom Sound (for Pomodoro):
       Download an `mp3` sound file (e.g., a bell or chime).
       Rename it to `bellring01.mp3`.
       Place it in the same directory as your other files.
       (If you skip this, the timer will use a simple browser `alert()`.)

4.Open `index.html` in your web browser:
       Doubleclick `index.html` in your file explorer, or drag and drop it into your browser.

That's it! The application should load in your browser, ready for use.
