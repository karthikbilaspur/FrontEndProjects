# All-in-One Calculator Hub

This project is a web-based calculator hub designed to provide various useful calculation tools in a single, easy-to-access interface. It's built with HTML, CSS, and JavaScript, focusing on clear functionality, a modern design, and responsiveness.

## Features

Calculator Selector:A dropdown menu allows users to easily switch between different calculators.
Age Calculator: Calculates a person's current age based on their birthdate.
Enhancements: Input validation (future dates), reset functionality.
BMI Calculator: Determines Body Mass Index with unit conversion and categorization.
Enhancements: Supports kg/lbs for weight, cm/inches for height, provides BMI category (Underweight, Normal, Overweight, Obese), reset functionality.
Tip Calculator: Helps calculate tip amount and total bill based on bill amount and tip percentage.
Enhancements: Quick-select buttons for common tip percentages (10%, 15%, 20%, 25%), input validation, reset functionality.
Multi-Shape Area Calculator: Calculates the area for different geometric shapes.
Enhancements:** Supports Circle, Square, Rectangle, and Triangle. Input fields dynamically change based on shape selection. Input validation, reset functionality.
Temperature Converter: Converts temperatures between Celsius, Fahrenheit, and Kelvin.
Enhancements: Real-time conversion as you type, validation for Kelvin (cannot be below absolute zero), reset functionality.
Detailed Descriptions: Each calculator section includes a brief description with a "Read More"/"Read Less" toggle for more information.
Smooth Scrolling: When a calculator is selected from the dropdown, the page automatically and smoothly scrolls to that calculator's section.

**Styling:** A clean, modern design using CSS with:
Root Variables: For easy customization of colors, fonts, and spacing.
Animations & Transitions: Subtle visual effects for elements like showing/hiding calculators and descriptions.
Media Queries: Ensures the layout is fully responsive and looks great on devices of all sizes (desktops, tablets, mobile phones).
SEO & Open Graph:Includes essential meta tags for search engine optimization and improved social media sharing.
Input Validation: Robust client-side validation to guide users and prevent incorrect input.
Error Highlighting:Invalid input fields are visually highlighted.
Clear/Reset Functionality: Each calculator has a dedicated reset button to clear inputs and results.
Dynamic Footer: Automatically displays the current year.

## Technologies Used

HTML5: For the structure and content of the web page.
CSS3:For styling and layout, including responsive design, animations, and transitions.
JavaScript (ES6+): For all interactive functionalities, calculations, and DOM manipulation.

## How to Set Up

To run this project locally, follow these simple steps:

1 Clone or Download the Repository:
    If using Git:

    Otherwise, download the ZIP file and extract it.

2.  Open `index.html`:
    Simply open the `index.html` file in your preferred web browser. You can do this by double-clicking the file, or by right-clicking and choosing "Open with" -> your browser.

    The application should load immediately, and you can start using the calculators.

## File Structure

The project has a straightforward file structure:

`index.html`: The main HTML file containing the structure of the calculator hub, including all calculator sections.
`script.js`: Contains all the JavaScript logic for calculator functionalities, input validation, "Read More/Less" toggles, and scrolling.
`style.css`: Contains all the CSS rules for styling the application, ensuring a consistent and responsive design.
