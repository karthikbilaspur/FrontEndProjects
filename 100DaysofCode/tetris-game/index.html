About This Game

This Advanced Tetris project is built entirely with vanilla JavaScript and HTML5 Canvas. It includes modern Tetris mechanics such as wall kicks, lock delay, combo scoring, hold system, and multiple gameplay modes.

Additional features include an AI autoplay system, replay recording and playback, mobile touch support, and accessibility-friendly controls.
