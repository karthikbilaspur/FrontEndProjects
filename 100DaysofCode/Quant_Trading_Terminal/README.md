# Quant Terminal Pro

A fully browser-based quantitative trading terminal built using pure HTML, CSS and JavaScript.

## Features

- Candlestick / Price chart
- RSI indicator
- Strategy engine
- Backtesting system
- Sharpe ratio calculation
- Heatmap visualization
- Equity tracking
- Dark UI
- Responsive layout

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js
- TwelveData Demo API

## How To Run

1. Download files
2. Open index.html in browser
Then open http://localhost:8000
