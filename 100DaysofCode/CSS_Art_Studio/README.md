# 🎨 CSS Art Studio

An interactive web application that allows users to create and customize stunning CSS shapes with various effects, filters, gradients, and animations. Design unique visual elements and instantly get the generated CSS code for your projects.

 ✨ Features

   Dynamic Shape Generation: Create a variety of geometric shapes including squares, rectangles, circles, rounded rectangles, triangles (up, down, left, right), pentagons, hexagons, octagons, stars, diamonds, parallelograms, and trapezoids.
   Extensive Customization:
       Colors & Gradients: Choose two primary colors and apply linear, radial, or conic gradients with adjustable directions.
       Transforms: Adjust borderradius and rotation in realtime.
       Filters: Apply CSS filters like saturate, contrast, sepia, grayscale, and huerotate with interactive sliders.
       Special Effects: Experiment with Glassmorphism, Neon Glow, Blob shapes, and repeating background patterns.
       Advanced Shadows: Enable multiple boxshadows with randomized properties for depth and visual flair.
       Animations: Add subtle animations like pulse or continuous rotation with adjustable duration.
       Layered Elements: Utilize pseudoelements (`::before`, `::after`) to create complex, layered visual effects with custom offsets and colors.
   Interactive Preview:
       Drag & Drop: Move your custom shape freely within the preview area.
       Zoom: Use your mouse wheel (or pinchtozoom on touch devices) to adjust the shape's scale.
       Realtime Feedback: See live updates of position and zoom percentage.
   Preset Management:
       Save your favorite designs to local storage with custom names.
       Load saved presets instantly.
       Delete unwanted presets from your list.
   Instant CSS Code Output: View and copy the exact CSS code generated for your unique design, including vendor prefixes for broader compatibility.
   Random Design Generator: Get inspired with a click of a button that generates a completely random design with varied shapes, colors, and effects.
   Accessibility & Usability:
       Semantic HTML structure and ARIA attributes for improved accessibility.
       Responsive design for use on various devices.
       Keyboard shortcuts for quick actions (R for Random, C for Copy CSS, X for Reset Position/Zoom, Z for Reset All Controls).
       Visual feedback for button actions (e.g., "Copied!").

 🚀 Getting Started

To run this project locally, simply save the three files (`index.html`, `style.css`, and `script.js`) into the same directory.

1.`index.html`: The main HTML structure.
2.  `style.css`: The styling for the entire application, including responsive design and animations.
3.  `script.js`: The core logic, handling all interactions, shape generation, effects, and preset management.

Then, open `index.html` in your web browser.

 🛠️ Technologies Used

   HTML5: For structuring the web page.
   CSS3: For styling, animations, transitions, and responsive design (using CSS Variables and Media Queries).
   JavaScript (ES6+): For all interactive functionality, DOM manipulation, and dynamic CSS generation.
