# BMI Calculator

A simple, interactive BMI (Body Mass Index) Calculator to help users track their body mass index, waist-to-hip ratio, and associated health risks. This application is built using HTML, CSS (with Bootstrap for styling), and JavaScript.

## Features

BMI Calculation:** Calculates BMI based on height and weight.

Age-Specific BMI Categories:** Uses appropriate BMI categories for adults and children (based on general guidelines).

Waist-to-Hip Ratio:** Calculates and displays the waist-to-hip ratio, providing additional health insights.

Health Risk Assessment:** Offers a basic assessment of chronic disease and cardiovascular disease risks.

Metric and Imperial Units:** Supports both metric (cm, kg) and imperial (ft/in, lbs) input units.

Multi-language Support:** Currently supports English and Spanish.

Date of Birth Picker:** Integrated `flatpickr` for easy date selection.

Dynamic Chart Visualization:** Displays BMI categories visually using Chart.js.

Share Results:** Allows users to share their results via the Web Share API (if supported by the browser).

Download PDF:** Users can download their results as a PDF document.

Responsive Design:** Styled with Bootstrap and custom CSS for a good experience across various screen sizes.

## Technologies Used

HTML5:Structure of the web application.
CSS3:Including a `style.css` for custom looks.
Bootstrap 5.1.3: Responsive framework for UI components.
JavaScript (ES6+):Core logic for calculations, form handling, and dynamic content.
Chart.js: For creating the BMI chart visualization.
Flatpickr: Lightweight date picker for the Date of Birth input.
jsPDF:For generating and downloading results as PDF files.
Font Awesome:** For icons (though not explicitly used in the provided snippets, it's linked).
