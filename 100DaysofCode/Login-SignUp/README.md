# Multi Form Types Project

A comprehensive showcase and demonstration of various modern web form designs and interactions. This project highlights different UI/UX patterns for login, registration, and data input, complete with responsive layouts, animations, and a unique feature to display the underlying HTML, CSS, and JavaScript code for each form.

## ✨ Features

*   **10 Diverse Form Types**: Explore a wide array of form designs, from simple to complex.
*   **Interactive Selection**: Easily switch between different form types using a dropdown menu.
*   **Detailed Descriptions**: Each form comes with a brief description of its design pattern and use case.
*   **Smooth Scrolling**: Navigate directly to the selected form section with a smooth scroll animation.
*   **Code Showcase**: For each form, view its complete HTML structure, relevant CSS styling, and associated JavaScript logic in an interactive modal, powered by Prism.js.
*   **Copy Code Functionality**: Quickly copy any displayed code snippet to your clipboard.
*   **Modern CSS**: Leverages CSS custom properties (variables), Flexbox, and Grid for maintainable and efficient styling.
*   **Animations & Transitions**: Enjoy a smooth user experience with subtle CSS transitions and animations on elements like form sections, inputs, buttons, and form steps.
*   **Responsive Design**: Optimized for various screen sizes, ensuring a consistent and pleasant experience on desktops, tablets, and mobile devices.
*   **Accessibility (A11y)**: Basic ARIA attributes implemented for better screen reader compatibility.
*   **SEO & Social Sharing**: Configured with meta tags, Open Graph, and Twitter Card data for improved search engine visibility and rich social media previews.

## 폼 유형 (Form Types)

Here's a list of the form types included in this project:

1.  **Simple Login with Icons**: Classic login with username/password and input field icons.
2.  **Login with Floating Labels**: Modern design where labels float above inputs on focus/fill.
3.  **Login & Registration Toggle**: Combines login and registration forms with a smooth toggle switch.
4.  **Responsive Registration Form**: A multi-column registration form that adapts gracefully to different screen sizes using CSS Grid.
5.  **Login with Social Media**: Standard login fields integrated with social media login options (Google, Facebook, GitHub).
6.  **Multi-Step Registration**: Breaks down a longer registration process into smaller, manageable steps with a visual progress indicator.
7.  **Form with Tooltips**: Inputs augmented with interactive tooltips for user guidance on hover/focus.
8.  **Password Visibility Toggle**: A login form featuring an eye icon to show/hide the password.
9.  **Search Bar with Suggestions**: An interactive search input that displays dynamic suggestions in a dropdown.
10. **Profile with Avatar Upload**: A profile setup form including an option to upload and preview a user avatar.

## ⚙️ Usage

1.  **Select a Form**: Use the dropdown menu at the top of the page to choose a form type.
2.  **View Description**: A description box will appear, explaining the selected form.
3.  **Scroll to Form**: Click the "More" button in the description to scroll smoothly to the form section.
4.  **Interact with Forms**: Experiment with the inputs, buttons, and unique features of each form.
5.  **View Code**: Click the "Show Code" button below any form to open a modal displaying its HTML, CSS, and JavaScript.
6.  **Copy Code**: Inside the code modal, click "Copy All" to copy all code snippets to your clipboard.
