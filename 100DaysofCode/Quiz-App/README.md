## üöÄ Project Overview

The **Interactive Quiz App** is a dynamic and engaging web application designed to test and expand users' knowledge across various subjects and difficulty levels. Built with modern HTML5, CSS3, and JavaScript, it offers a seamless and interactive learning experience with customizable quizzes, real-time feedback, and a sleek user interface.

## ‚ú® Features

* **Customizable Quizzes:**
* Select your preferred **Subject:** Math, History, Geography, or Science.
* Choose your desired **Difficulty:** Easy (20 Qs), Medium (15 Qs), or Hard (10 Qs).
* **Single-Question Display:** Questions are presented one at a time for focused engagement.
* **Per-Question Timers:** Time limits for each question (20s Easy, 30s Medium, 40s Hard) add a challenge.
* **Quiz Navigation:** Move between questions using "Previous" and "Next" buttons, allowing review and changes before final submission.
* **Real-time Results:** Get your score instantly after completing the quiz.
* **Detailed Explanations:** Collapsible accordion sections provide correct answers and explanations for each question after quiz submission.
* **Dark Mode/Light Mode Toggle:** Switch between themes for a personalized viewing experience, with preference saved locally.
* **Content Carousel:** Discover related articles and learning resources in an interactive carousel.
* **Responsive Design:** Optimized for a consistent and visually appealing experience across desktops, tablets, and mobile devices.
* **Accessibility (ARIA):** Enhanced with ARIA attributes and keyboard navigation support for screen readers and users with disabilities.
* **Robust Error Handling:** Informative console logging and user-facing messages for missing DOM elements or invalid selections.

## üìù Usage

1. **Start Quiz Selection:** Click the "Start Quiz Selection" button on the description page.
2. **Choose Your Quiz:** Select your desired difficulty (Easy, Medium, Hard) and subject (Math, History, Geography, Science).
3. **Start Quiz:** Click "Start Quiz" to begin the challenge.
4. **Answer Questions:** Each question will appear with a timer. Select your answer and click "Next" to proceed.
5. **Navigate:** Use "Previous" and "Next" buttons to move between questions.
6. **Submit Quiz:** On the last question, the "Next" button will change to "Submit Quiz". Click it to finish.
7. **Review Results:** Your score and detailed explanations will be displayed.
8. **Restart:** Use the "Restart Quiz" button in the header to go back to the selection screen.
9. **Toggle Theme:** Use the "Dark Mode" / "Light Mode" button in the header to switch themes.
